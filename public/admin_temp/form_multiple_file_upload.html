Note

                                    
File Upload widget with multiple file selection, drag&drop support, progress
                                        bars, validation and preview images, audio and video for jQuery.
Supports
                                        cross-domain, chunked and resumable file uploads and client-side image resizing.
Works
                                        with any server-side platform (PHP, Python, Ruby on Rails, Java, Node.js, Go
                                        etc.) that supports standard HTML form file uploads.
